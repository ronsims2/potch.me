box-sizing FTW…

Let's create two floated columns of content each half the page width. Let's keep the content from touching with a padding, and add a nice border too. Here's some nice semantic CSS: 

    .row {
      overflow: hidden;
    }

    .col {
      width: 50%;
      float: left;
    }

    .content {
      border: 10px solid blue;
      padding: 1em;
    }

Without box-sizing

Here's our markup!

    <section class="row">
      <div class="col content">
        Lorem ipsum…
      </div>
      <div class="col content">
        Lorem ipsum…
      </div>
    </section>

With box-sizing

Using the original markup:

    <section class="row">
      <div class="col content">
        Lorem ipsum…
      </div>
      <div class="col content">
        Lorem ipsum…
      </div>
    </section>

And some new CSS:

    .col {
      box-sizing: border-box;
      -moz-box-sizing: border-box;
      -webkit-box-sizing: border-box;
    }

and voila!
